This Banking App is a basic Java application designed to simulate common banking operations. It demonstrates object-oriented programming principles such as classes, objects, inheritance, and encapsulation. The project structure follows a standard Maven-style layout with src/main/java and src/main/resources.

The application can serve as a starter template for learning Java backend development or extending into a full financial application


Features:

Create bank accounts

Deposit and withdraw funds

Check account balances

Basic error handling (e.g., insufficient funds)

Organized project structure for easy extension
